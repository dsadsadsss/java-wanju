#### 参数:vls,vms,rel,3x含义:

vls即vless+ws+tls

vms即vmess+ws+tls

rel即reality

hy2或hys,即hysteria2

tuic,字面意思

3x即3种协议，包括vmess,hy2,tuic

xhttp即xhttp，需要使用固定隧道，网络打开grpc

sock即socks5,会在list.log里同时生成TG格式和节点格式

注意:DOCKER部署时只能设置vms或vls或xhttp
